# Qonyx

Qonyx is an AI quantitative trading command center with a premium polished onyx
interface. The current app is a Vite + React implementation of the Onyx
Terminal design system, using Logo 1 as the main app mark and the mascot asset
for onboarding, AI research, and friendly empty states.

## Run locally

```bash
npm install
npm run dev
```

## Verify

```bash
npm run build
```

## Structure

```txt
apps/web
  public/brand        Brand images, favicon, and social preview assets
  src                 React application and Onyx Terminal components
```
